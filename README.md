# Hangman (Виселица) — консольная игра на Java
## Описание проекта

Данный проект представляет собой реализацию классической игры «Виселица» на языке программирования Java с консольным интерфейсом.

Игра запускается в терминале и позволяет пользователю угадывать скрытое слово по буквам, имея ограниченное количество попыток. Проект разработан в учебных целях и демонстрирует работу с базовыми конструкциями Java: ввод/вывод, строки, коллекции, циклы и условные операторы.

Правила игры основаны на классической версии, подробно описанной на Википедии.
